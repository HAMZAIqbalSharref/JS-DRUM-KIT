# JavaScript30

This repository contains my implementations of the JavaScript30 challenge by Wes Bos. JavaScript30 is a 30-day vanilla JavaScript coding challenge that focuses on building projects without any libraries or frameworks, emphasizing DOM manipulation and modern JavaScript features.

## Projects

### Day 1: JS Drum Kit
A simple drum kit web application built with JavaScript, HTML, and CSS.

- Play drum sounds by pressing keys on your keyboard
- Visual feedback when keys are pressed
- Responsive design

## Goals

- Practice DOM manipulation
- Learn vanilla JavaScript
- Build fun web applications

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
