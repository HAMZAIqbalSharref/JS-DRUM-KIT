# JS Drum Kit

This project is built as part of JavaScript30, a 30-day vanilla JavaScript coding challenge by Wes Bos. It focuses on DOM manipulation and event handling to create a fun drum kit interface.

## Features

- Play drum sounds by pressing keys on your keyboard
- Visual feedback when keys are pressed
- Responsive design

## How to Use

1. Open `index.html` in your web browser
2. Press the keys A, S, D, F, G, H, J, K to play different drum sounds
3. Enjoy making music!

## Files

- `index.html`: The main HTML file
- `app.js`: JavaScript code for handling key presses and playing sounds
- `style.css`: CSS styles for the drum kit interface
- `sounds/`: Folder containing the drum sound files

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
